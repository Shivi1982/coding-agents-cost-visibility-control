Session ↔ Merge Request Linkage

Sessions matched to MRs via commit SHA — cost attribution per merge request

📡 Load Data from CloudWatch
fields @timestamp, @message | filter @message like /session_commit_link/ | parse @message '{"session_id":"*","commit_sha":"*","mr_id":"*","mr_title":"*","developer":"*","cost_usd":*}' as session_id, commit_sha, mr_id, mr_title, developer, cost_usd | stats sum(cost_usd) as total_cost, count(distinct session_id) as sessions | by mr_id, mr_title, developer | sort total_cost desc
